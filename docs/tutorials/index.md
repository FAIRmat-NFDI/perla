## Jupyter Notebooks

These are a set of notebooks that performed analysis that we describe in the PERLA paper. They require a parquet file that you need to generate with the [Query PERLA Database](../notebooks/query-perovskite-database.ipynb) notebook or by creating it NOMAD directly (TODO).

- **[Query PERLA Database](../notebooks/query-perovskite-database.ipynb)** - Learn how to retrieve data programmatically via the NOMAD API
- **[Performance Evolution Analysis](../notebooks/performance-evolution.ipynb)** - Analyze power conversion efficiency trends over time
- **[Bandgap Evolution Analysis](../notebooks/bandgap-evolution.ipynb)** - Explore bandgap and composition trends in the literature
- **[Architecture Evolution Analysis](../notebooks/architecture-evolution.ipynb)** - Track device architecture trends (n-i-p vs p-i-n structures)
- **[Diversity Analysis](../notebooks/diversity-analysis.ipynb)** - Examine dataset composition and diversity metrics
- **[ML Prediction with CrabNet](../notebooks/crabnet-perovskite-bandgap-prediction.ipynb)** - Tutorial on machine learning predictions using perovskite data
- **[Distribution Shift Case Study](../notebooks/ml-distribution-shift-case-study.ipynb)** - Analyze distribution shifts in ML models
- **[Papersbot Pipeline](../notebooks/perovskite-paperbot-plot.ipynb)** - Visualize the automated extraction pipeline
- **[Performance Check Filtering](../notebooks/physics_filter.ipynb)** - Apply physics-based validation filters to data
- **[PERLA Evaluations Analysis](../notebooks/perla-evals-analysis.ipynb)** - Analyze the performance of PERLA's data extraction evaluations
