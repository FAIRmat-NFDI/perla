# Ecosystem & Credits

PERLA is built through collaborative efforts across multiple institutions and relies on a rich ecosystem of tools, platforms, and data sources.

## Core Team & Contributors

### Lead Institutions
- **FAIRmat-NFDI**: NFDI consortium for research data on condensed matter physics and chemistry
- **NOMAD Laboratory**: Materials science data platform and infrastructure
- **LAMA Lab**: Laboratory for Advanced Materials Research

### Key Contributors
PERLA development involves researchers from computational materials science, machine learning, and photovoltaics communities. The project benefits from ongoing collaboration between domain experts and data scientists.

## Technology Ecosystem

### Platforms & Infrastructure

**NOMAD Platform**
- Core data infrastructure and FAIR data principles
- Schema definitions for perovskite solar cell data
- API infrastructure and data access protocols
- Metadata management and provenance tracking

**Large Language Models**
- State-of-the-art NLP models for text analysis
- Multimodal processing capabilities
- Continuous model improvements and updates
- Physics-aware fine-tuning for materials science
