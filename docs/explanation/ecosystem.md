# Ecosystem & Credits

PERLA is built through collaborative efforts and relies on a rich ecosystem of tools, platforms, and data sources.

## Core Team & Contributors

### Lead Institutions
- [**FAIRmat-NFDI**](https://www.fairmat-nfdi.eu/): NFDI consortium for research data on condensed matter physics and chemistry
- [**LAMALab**](https://lamalab.org): Lab for AI in Materials Science at FSU Jena/HIPOLE Jena

## Repository Structure

### Main Repositories

**[FAIRmat-NFDI/perla](https://github.com/FAIRmat-NFDI/perla)**

- Main project repository
- Documentation and tutorials

**[lamalab-org/perla-extract](https://github.com/lamalab-org/perla-extract)**

- Data extraction pipeline
- LLM integration and processing tools
- Bot system

**[FAIRmat-NFDI/nomad-perovskite-solar-cells-database](https://github.com/FAIRmat-NFDI/nomad-perovskite-solar-cells-database)**

- NOMAD schema definitions
- Database normalization procedures
- Example datasets and validation scripts
- Halide perovskite ions database and processing

## Community & Collaboration

### Open Science Principles
PERLA is committed to open science and follows FAIR data principles:

- **Open Source**: All code and tools freely available
- **Open Data**: Database accessible without restrictions
- **Open Standards**: Community-driven schema development
- **Open Collaboration**: Transparent development process

### Citation & Attribution

When using PERLA data or tools, please cite it as described in the [citation guidelines](citation.md)
